# PNC Retail Banking Performance Monitor

Quarterly variance analysis, rolling forecasts, and an executive dashboard built on PNC's real reported segment data.

## Data Source
PNC Quarterly Earnings Releases: Q1 2025, Q2 2025, Q3 2025, Q1 2026
Available at investor.pnc.com

## Skills Demonstrated
- Variance analysis and budget vs. actual reporting
- Continuous quarterly forecasting cycle
- Metric-driven performance dashboards
- Financial reporting for senior management
- Audit-ready data exports

## Tech Stack
Python, Pandas, NumPy, SciPy, Plotly, Streamlit

## How to Run Locally
streamlit run app.py
